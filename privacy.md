Welcome to RoTracker.

This Privacy Policy explains how RoTracker collects, uses, and protects information when you use the Service.

## 1. Information We Collect

RoTracker may collect and process the following data:

### a) Discord Data
- Server (Guild) IDs
- Channel IDs
- Role IDs
- User IDs (for command execution & permissions)
- Message IDs (for dashboard updates)
- Configuration settings

### b) Usage Data
- Commands used
- Error logs
- Diagnostic information

### c) Roblox Data
RoTracker accesses publicly available Roblox data via official APIs:
- Place IDs / Universe IDs
- Game metadata (title, description, stats)
- Badges, gamepasses, developer products, assets

We do **not** collect Roblox account credentials.

## 2. How We Use Information

Collected data is used strictly for:

- Operating bot functionality
- Maintaining trackers & dashboards
- Delivering notifications
- Preventing abuse / debugging
- Improving Service reliability

We do not sell or rent your data.

## 3. Data Storage & Retention

- Data is stored only as long as necessary
- Server configuration persists until removal
- Data is deleted when trackers are removed or upon request

## 4. Data Sharing

We do not share data except:

- When required by law
- To protect Service integrity
- With infrastructure providers (hosting, database)

## 5. Security

We implement reasonable safeguards, including:

- Access control
- Secure storage practices
- Limited data collection

However, no system is 100% secure.

## 6. User Rights

Server administrators may:

- Remove stored configuration
- Request data deletion
- Stop using the Service at any time
